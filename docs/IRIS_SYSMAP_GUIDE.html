08 Deployment & Persistence
Pi4 Overlayfs Persistence Pattern
Pi4 — required for every file deploy
sudo mount -o remount,rw /media/root-ro
sudo cp /home/pi/<file> /media/root-ro/home/pi/<file>
sudo chown pi:pi /media/root-ro/home/pi/<file>
sudo chmod 644 /media/root-ro/home/pi/<file>
sync
sudo mount -o remount,ro /media/root-ro
md5sum /home/pi/<file> /media/root-ro/home/pi/<file>
# md5 must match — no exceptions
Never use overlayroot-chroot — MD5 mismatch confirmed Batch 1A. SFTP: write to /tmp first, then sudo cp.
