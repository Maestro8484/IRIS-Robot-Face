## 1. PROJECT OVERVIEW

Stationary interactive AI robot face. Two round TFT displays as eyes, single-axis pan servo head for face tracking, voice pipeline via Pi4 Jarvis assistant. Wall-E aesthetic -- purple/pink acrylic eye housings, blue glitter acrylic base, laser-cut construction.

### Goals
- Eyes animate with realistic motion (TeensyEyes engine)
- Eyes express emotions driven by Jarvis AI responses
- Head pans (1 axis only) to track faces via Person Sensor (Pico system)
- Face detection triggers Jarvis greeting via Pi4
- Megan's complaint addressed: robot does NOT stare unless actively in conversation
- Voice sounds natural, JARVIS-like (not robotic Piper TTS)

## 2. FINAL PHYSICAL ARCHITECTURE

### Base (fixed, stationary)
```
┌─────────────────────────────────────────────┐
│  Pi4 (flat/horizontal, rear of base)        │
│   └── short 40-pin ribbon (~10cm)           │
│        └── ReSpeaker 2-Mic HAT             │
│             mics via LEFT+RIGHT side walls  │
│             12mm holes + stainless mesh     │
│                                             │
│  Pi camera -- fixed, front face cutout      │
│               angled slightly upward        │
│               CSI → HDMI adapter cable      │
│               (Arducam CSI-to-HDMI kit)     │
│                                             │
│  Speaker (1x mono 2W) -- front face grille  │
│  Eyes Teensy 4.0 -- USB to Pi4 USB 3.0     │
│  Raspberry Pi Pico -- VSYS Pin 39 from      │
│                        Pi4 5V GPIO Pin 2/4  │
│  SG90 servo -- Pi4 5V GPIO Pin 2/4 power   │
│                                             │
│  Power in: USB-C 3A minimum supply         │
└─────────────────────────────────────────────┘
```

### Rotating Head (PAN ONLY -- 1 axis)
```
┌─────────────────────────────────────────────┐
│  2x GC9A01A TFT displays (240x240 round)    │
│  Person Sensor -- center bridge             │
│  Pan servo -- GPIO 0 on Pico               │
│  Wires down neck to base                   │
└─────────────────────────────────────────────┘
```

### Key Architecture Decisions
- ReSpeaker HAT soldered mics -- cannot separate, stays with Pi4 in base
- Pi camera fixed in base -- ribbon fatigue risk on rotating mount
- Arducam CSI-to-HDMI adapter for durable camera cable routing
- Camera angled slightly upward -- covers adults + Leo (9) + Mae (5)
- Single mono 2W speaker sufficient (TTS is mono)
- Servo system is Raspberry Pi Pico NOT Teensy 4.0
- PAN ONLY -- no tilt servo, single axis rotation
- Pico powered via VSYS pin 39 -- no micro USB needed
- SG90 pan servo powered from Pi4 5V GPIO (28AWG, 4-5" run fine)
- Eyes Teensy + 2 displays on Pi4 USB 3.0 (~220mA, within 900mA budget)

### Power Budget
| Device | Draw |
|--------|------|
| Pi4 + ReSpeaker HAT | ~1.3A |
| Pi camera | ~250mA |
| Eyes Teensy + 2x TFT | ~220mA |
| Pico + Person Sensor | ~35mA |
| SG90 pan servo | ~200mA typical, 500mA peak |
| **Total** | **~2.0A typical** |
3A supply = ~1A headroom. Comfortable.

```python
import serial, threading, re

teensy = None
try:
    teensy = serial.Serial('/dev/ttyACM0', 115200, timeout=0.1)
except Exception as e:
    print(f"[EYES] Serial not available: {e}")

last_face_time = 0
FACE_COOLDOWN = 30

def send_emotion(emotion):
    if teensy:
        try:
            teensy.write(f"EMOTION:{emotion}\n".encode())
        except Exception:
            pass

EMOTION_KEYWORDS = {
    'happy': 'HAPPY', 'excited': 'HAPPY', 'great': 'HAPPY',
    'curious': 'CURIOUS', 'interesting': 'CURIOUS', 'wonder': 'CURIOUS',
    'angry': 'ANGRY', 'frustrated': 'ANGRY',
    'sleepy': 'SLEEPY', 'tired': 'SLEEPY',
    'surprised': 'SURPRISED', 'wow': 'SURPRISED',
    'sad': 'SAD', 'sorry': 'SAD',
}

def extract_and_send_emotion(text):
    match = re.search(r'\[EMOTION:(\w+)\]', text)
    if match:
        send_emotion(match.group(1))
        return re.sub(r'\[EMOTION:\w+\]\n?', '', text).strip()
    for keyword, emotion in EMOTION_KEYWORDS.items():
        if keyword in text.lower():
            send_emotion(emotion)
            return text
    send_emotion('NEUTRAL')
    return text

def teensy_listener():
    global last_face_time, in_conversation
    while True:
        try:
            if teensy and teensy.in_waiting:
                line = teensy.readline().decode().strip()
                if line == "FACE:1":
                    now = time.time()
                    if not in_conversation and (now - last_face_time) > FACE_COOLDOWN:
                        last_face_time = now
                        send_emotion("CURIOUS")
                        speak("Oh, hello. I see you.")
                elif line == "FACE:0":
                    send_emotion("NEUTRAL")
        except Exception:
            pass
        time.sleep(0.05)

threading.Thread(target=teensy_listener, daemon=True).start()
```
