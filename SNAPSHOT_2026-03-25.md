## 6. SERIAL PROTOCOL

**Pi4 → Teensy:**
```
EMOTION:NEUTRAL\n     -- idle, autoMove on
EMOTION:HAPPY\n
EMOTION:CURIOUS\n
EMOTION:ANGRY\n       -- flame eyes, 9s auto-revert
EMOTION:SLEEPY\n
EMOTION:SURPRISED\n
EMOTION:SAD\n
EMOTION:CONFUSED\n    -- hypnoRed eyes, 7s auto-revert
EYES:SLEEP\n          -- both displays black, rendering halted
EYES:WAKE\n           -- restore current eye, apply NEUTRAL
EYE:n\n               -- switch default eye to index n (0-8), persists across emotions
MOUTH:x\n             -- set mouth expression (NEUTRAL/HAPPY/ANGRY/SAD/SURPRISED/CURIOUS/SLEEPY/SLEEP)
```

**Teensy → Pi4:**
```
FACE:1\n    -- face detected (30s cooldown)
FACE:0\n    -- face lost
```
