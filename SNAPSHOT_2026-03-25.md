## 6. SERIAL PROTOCOL

**Pi4 → Teensy:**
```
EMOTION:NEUTRAL\n     -- idle, autoMove on
EMOTION:HAPPY\n
EMOTION:CURIOUS\n
EMOTION:ANGRY\n       -- flame eyes, 9s auto-revert
EMOTION:SLEEPY\n
EMOTION:SURPRISED\n
EMOTION:SAD\n
EMOTION:CONFUSED\n    -- hypnoRed eyes, 7s auto-revert
EYES:SLEEP\n          -- sleep renderer starts, mouth dims + snore animation
EYES:WAKE\n           -- restore current eye, apply NEUTRAL, restore mouth intensity
EYE:n\n               -- switch default eye to index n (0-6)
MOUTH:x\n             -- set mouth expression (0=NEUTRAL ... 8=SLEEP/OFF)
```

**Teensy → Pi4:**
```
FACE:1\n    -- face detected (30s cooldown) -- also fires during sleep
FACE:0\n    -- face lost
```
